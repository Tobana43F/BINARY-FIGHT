### ゲーム概要

**ストリートファイターシリーズ** を参考に開発した2D格闘ゲームです。

近距離でのインファイトがメインのシステムとなっており、シンプルながらも駆け引きのあるゲームプレイを目指しました。

このゲームでの一番の要素は **足払い** です。  
足払いで差仕返し、その後の起き攻めをいかにして行うかが勝敗を分けます。
